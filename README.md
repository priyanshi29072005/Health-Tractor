# Health-Tractor
# Overview of the project
This project is a simple command-line application to help users track their daily health habits and calculate a "health score" based on wellness report . It prompt users for daily data such as sleep, water intake, protein, screen time, steps, and meals, then generates a health report and personalized feedback based on your score.

# Features
Tracks six important daily health habits

Calculates an overall health score with easy-to-understand ratings

Provides daily summary and suggestions based on performance

Simple interface suitable for beginners and easy customization in easy level feedback.
